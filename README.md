
# Qampus Web Application

A web application for managing cashier and payment transactions.

## Overview

Qampus is a simple payment processing system that allows cashiers to manage transactions and users (payers) to complete payments without requiring login credentials.

## Features

- **Cashier Dashboard**: Process and manage transactions
- **Payor Interface**: Make payments easily
- **No Login Required**: Both cashiers and payers can access the system without authentication
